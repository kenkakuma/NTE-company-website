## 🏗️ 技术架构详情

### 前端技术栈
- **核心框架**: Vue 3 (Composition API)
- **构建工具**: Vite 2.9.9
- **UI框架**: Material Design + Bootstrap 5
- **样式系统**: SCSS + CSS Variables
- **组件库**: Vue Material Kit 2定制版
